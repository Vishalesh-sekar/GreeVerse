# ✨ Overview

**GreeVerse** is a next-generation, lightweight gallery application designed to organize and manage your local media in one place.

Unlike traditional gallery applications, GreeVerse aims to combine:

* 📷 Photo Gallery
* 🎥 Video Player
* 🎵 Music Player
* 📄 Document Viewer

inside one beautiful application while remaining completely offline and privacy-focused.

No cloud.

No ads.

No tracking.

Just your files.
